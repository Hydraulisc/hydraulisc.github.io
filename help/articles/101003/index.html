Getting Started

Creating your first account on Hydraulisc

Important!

Please note that this is an extremely basic guide, designed for newbies, with the aim of solving some minor login issues.


How to Create your Account

Firstly, make sure you are on the official Hydraulisc App/Site! This guide covers the official instance only!

  1. Head to https://hydraulisc.net/register.
  2. Fill in the fields as required.
    • Usernames can only contain alphanumeric characters and periods ('.')
    • Usernames must be between 3 and 30 characters.
    • We recommend that you use a strong password, 8-24 characters with a capital, lowercase, and symbol.
  3. Upon clicking the "Register" button, you should be on an "Account Created Successfully" page, which gives you a username followed by a discriminator! This is very important:
    • If you use a password manager, update your username to the one displayed on this page, with the '#' and four digits that follow it. This is your login, not your display name - you can change that later.
    • If you don't use a password manager, it is very important that you write down the suffix to your login somewhere safe, as you cannot login or recover your account if you lose this suffix!
  4. OPTIONAL - You can click the 'Return to Login' button on this page to test your password manager and make sure it's updated, or to check you wrote down the suffix correctly.
  5. OPTIONAL - You can update various account details from this page, including
    • Profile Image
    • Profile Banner
    • Default Theme
    • Language Preference
    • Short Biography

If you have issues with any of the steps mentioned above, feel free to request additional support in our Discord.

Please note, these instructions work on all Official Hydraulisc Instances.
