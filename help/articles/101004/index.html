Creating Posts

Creating your first posts on Hydraulisc

Important!

NOTICE: As per Hydraulisc Guidelines, you should only post content that you've taken or have the right to share!

Also note that this is an extremely basic guide, designed for newbies, with the aim of aiding in platform growth.


Posts Manager

Firstly, make sure you are on an official Hydraulisc App/Site, and logged in.

As per always, this guide covers official instances only!

It is recommended to have JavaScript enabled for the best posting experience!

  1. Navigate to the post creation section. If you get redirected to the login page, login and try again.
  2. OPTIONAL - Add a post title/header or short description. We give you up to one thousand (1000) characters to caption your post. Don't forget to #hash your creations!
  3. Select a file to post. You can't upload nothing, of course.
    • PNG, JPEG, JPG, GIF & WEBP formats are supported.
    • 25Mb file size hard limit.
    • Grid posts/multi posts currently in beta!
  4. You should get a preview of the image you selected.
  5. Review the data you've enetered. If it's all done:
  6. Upload!

If you have issues with any of the steps mentioned above, feel free to request additional support in our Discord.

Please note, these instructions work on all Official Hydraulisc Instances.
